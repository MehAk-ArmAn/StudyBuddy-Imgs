# StudyBuddy-Imgs

External public image asset repository for StudyBuddy.fun.

Use raw URLs in the website database:

`https://raw.githubusercontent.com/MehAk-ArmAn/StudyBuddy-Imgs/main/<path>`

Main folders:
- `brand/`
- `hero/`
- `apps/`
- `homepage-paths/`
- `dashboard/`
- `decor/`

The Laravel website should store only URL/path strings in the database. Image binaries stay here so the website repo remains lightweight.
